# Installation Guide

This document describes how to set up and install the **ID Vision** facial detection system.

---

## 1️⃣ Requirements

- Operating System: Ubuntu / Debian 12
- Python 3.10+
- InsightFace model installed
- Camera(s) connected and accessible

---

## 2️⃣ Steps

1. Clone the private repository:
   ```bash
   git clone https://github.com/Qasemt/id-vision-private.git
   cd id-vision-private
   ```
